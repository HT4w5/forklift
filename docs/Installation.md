# Installation

## Prerequisites
- [Sing-box](https://github.com/SagerNet/sing-box) binary. (Not necessarily in $PATH but recommended.)

## Pre-compiled
Visit [Github releases](https://github.com/HT4w5/forklift/releases) and download latest forklift binary for your platform.

## Building
1. Install `make`.
1. Clone repository.
1. Run `make`.
